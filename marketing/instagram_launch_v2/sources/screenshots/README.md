# Optional device screenshots

Drop **PNG** exports of the real apps here to replace the synthetic map/driver/rider composites in a future iteration of `scripts/build_ad_creatives.py`.

Suggested names:

| File | Use |
|------|-----|
| `rider_map.png` | Rider map / tracking surface |
| `rider_booking.png` | Destination / home strip |
| `driver_incoming.png` | Driver offer / accept card |

Keep **2:1 or taller** portrait crops for phone frames; minimum width **900px** for sharp composites.
